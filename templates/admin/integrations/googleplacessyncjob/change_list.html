    Google Places Hotel Import - How to Use

    Setup (One-time)

    1. Get a Google Places API key from Google Cloud Console
    2. Enable "Places API (New)" in your Google Cloud project
    3. Add GOOGLE_PLACES_API_KEY to your Railway environment variables
    4. Click "Test API Connection" to verify it works

    Workflow

    1. Browse Hotels - Click the green button above
    2. Select a City or enter a custom search query
    3. Review Hotels - See photos, ratings, and addresses
    4. Select Hotels - Check the boxes for hotels you want to import
    5. Import - Creates draft listings with photos and amenities
    6. Add Klook Links - Search each hotel on Klook and add affiliate URL

    Notes

    • Hotels are imported as Draft listings
    • Photos are downloaded automatically (can be disabled)
    • Amenities are extracted as text from Google data
    • Already imported hotels are grayed out
