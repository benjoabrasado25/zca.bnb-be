    Amadeus Hotel Sync - How to Use

    Setup (One-time)

    1. Add AMADEUS_API_KEY and AMADEUS_API_SECRET to your Railway environment variables
    2. Click "Test API Connection" to verify credentials work

    Workflow

    1. Browse Hotels - Click the green button above to start
    2. Select a City - Choose from Philippine cities (Manila, Cebu, etc.)
    3. Review Hotels - See all hotels from Amadeus for that city
    4. Check Klook - Click "Search" link next to each hotel to check if it exists on Klook
    5. Select Hotels - Check the boxes for hotels you want to import
    6. Import - Click "Import Selected Hotels" to create draft listings
    7. Add Affiliate Links - Go to Listings, find the imported hotels, add Klook affiliate URL manually

    Notes

    • Hotels are imported as Draft listings with minimal data (no images)
    • You must manually add Klook affiliate URLs after verifying the hotel exists on Klook
    • Already imported hotels are grayed out and cannot be re-selected
    • Sync jobs are logged below for reference
